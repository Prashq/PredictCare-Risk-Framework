# PredictCare-Risk-Framework

An AIML-based web application that predicts diabetes risk using patient health data.

# Features
- Diabetes risk prediction (Low / Medium / High)
- Patient data entry system
- Dashboard with search functionality
- Risk history visualization (graph)
- AI-based reasons & advice

# Technologies Used
- Python (Flask)
- Machine Learning (Random Forest)
- HTML, CSS, JavaScript
- SQLite
- Chart.js

# How to Run
1. Install required libraries
2. Run `python app.py`
3. Open http://127.0.0.1:5000

# Dataset
- Pima Indians Diabetes Dataset
